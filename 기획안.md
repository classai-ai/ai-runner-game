# 게임 소개

플레이어는 AI 캐릭터가 되어  
사이버 세상을 달리며 바이러스를 피하고,  
백신을 획득하여 생존하면서 최대한 높은 점수를 기록하는 게임이다.

게임 방식은 템플런, 서브웨이 서퍼와 같은  
**3레인 무한 러닝 게임**이다.

---

# 세계관

AI들이 살아가는 사이버 월드.

최근 바이러스들이 시스템을 공격하면서  
AI 세계가 붕괴되기 시작했다.

플레이어는 살아남기 위해  
끊임없이 앞으로 달리며

- 바이러스를 피하고
- 백신을 수집하며
- 데이터를 획득해 점수를 올린다.

---

# 플레이 방식

게임은 자동으로 앞으로 이동한다.

플레이어는

- 왼쪽
- 가운데
- 오른쪽

세 개의 레인을 이동한다.

조작은 매우 단순하다.

### 이동

← : 왼쪽

→ : 오른쪽

또는

A / D

또는 모바일

왼쪽 스와이프

오른쪽 스와이프

---

### 점프

Space

또는 위 스와이프

장애물을 뛰어넘는다.

---



### 슬라이딩(선택사항)

↓

또는 아래 스와이프

낮은 장애물을 피한다.

---

# 추천 개발 환경

- **게임 엔진:** Unity (3D) 
- **언어:** C# 
- **AI 바이브 코딩:** Cursor, GitHub Copilot, ChatGPT를 활용한 프롬프트 기반 개발 
- **타깃 플랫폼:** PC(키보드) → 이후 모바일(Android/iOS) 확장

# MVP(최소 기능 버전)

1. 메인 메뉴
2. 캐릭터 선택(병아리/흰곰)
3. 3레인 자동 달리기
4. 좌우 이동
5. 점프
6. 무한 맵 생성
7. 바이러스 장애물
8. 백신 아이템(목숨 +1, 최대 5개)
9. 5개의 목숨 시스템 및 무적 시간
10. 점수 및 거리 표시
11. 게임 오버 및 재시작
