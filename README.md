# AI Runner : Escape from Virus

사이버 세상을 달리며 바이러스를 피하고 백신을 획득하는 HTML5 러너 게임입니다.

## 실행 방법

1. `index.html` 파일을 웹 브라우저로 열기
2. 또는 Live Server 등 로컬 서버로 실행

## 조작법

- **← / →** 또는 **A / D** : 좌우 이동
- **스페이스** : 점프

## 프로젝트 구조

```
├── index.html      # 메인 HTML
├── game.js         # 게임 로직
├── audio.js        # 오디오 관리
├── style.css       # 스타일
├── assets/
│   ├── audio/      # BGM, 효과음
│   └── sprites/    # 캐릭터 스프라이트
└── 기획안.md       # 게임 기획 문서
```

## 기술 스택

- HTML5 Canvas
- Vanilla JavaScript
- CSS3
